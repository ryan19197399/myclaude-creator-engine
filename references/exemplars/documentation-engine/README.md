# Documentation Engine

> Extract, generate, and validate documentation from codebases.

## What It Does

A multi-component system with 3 parts:
- **quick-doc** (skill) — Fast README generation
- **doc-analyst** (agent) — Deep architecture analysis
- **full-suite** (workflow) — Complete documentation pipeline

## Installation

```bash
myclaude install documentation-engine
```

## Usage

```
/documentation-engine                    # Assess and recommend
/documentation-engine "generate readme"  # Quick README
/documentation-engine "full suite"       # Complete docs
```

## Requirements

- Claude Code >= 1.0.0
- Read + Glob + Grep tool access
